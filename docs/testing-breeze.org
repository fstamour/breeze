#+title: Testing Breeze

* Summary of all the tests

- common lisp unit tests
  - ~make test~
  - for each common lisp implementation:
    - ~guix-shell-test-*~
    - ~guix-container-test-*~
- emacs lisp unit tests
- emacs integration tests
- publish the org-mode documents
  - ~make public~
- build the executable

* How to run the tests

#+begin_src lisp
(ql:quickload "breeze/test")
(asdf:test-system "breeze")
#+end_src

Or from the command line:

#+begin_src shell
./scripts/test-sbcl.sh
#+end_src

OR

#+begin_src shell
make test
#+end_src
